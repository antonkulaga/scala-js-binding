Scala-JS-Binding

is a scalaJS
framework that lets you:

Bind html elements
and properties directly to variables of ScalaJS classes
Handle events and changes as flows
Scala.Rx is used to create flow graphs
and manipulate with them in functional ways
Create reusable controls with views
You can create scalajs view classes that can bind
to hml elements and organize themselves in trees
Bind to LinkedData
Scala-JS-Binding supports banana-rdf and RDFa,
so you can bind to nodes of RDF graphs

Greetings binds to a say variable that depends on two other variables, just change them and immediately see the result!
