scala-js-binding supports binding to collections of elements. By the way, all menus at this website/examples are done this way. Let's look at some examples.

The most common example of collection is the menu where each item looks similar and is loaded from some collection of elements. Here you can see an HTML code required for the menu to appear.

Apply template

There data-param-path makes a request to corresponding address and loads a menu into the page. In current example there is a tag with data-template="true" this tag is a template for each item that will be created.

Each item has data-load-into attribute. This attribute defined where the data should be loaded after menu is clicked. Usually data is loaded with ajax into some html element to avoid whole page refreshing.

You can edit template on the right and push apply template and the template will be rerendered according new template
